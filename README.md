## Summary
**﻿Sản phẩm nhóm - Phần mềm quản lý công việc khách sạn Tasmana** dành cho chương trình **BCMP Korea**, thuộc khóa học **Kỹ thuật Phần mềm**.

**Chức năng chính:**
- Quản lý nhân viên, nhiệm vụ của nhân viên, các nhóm và phòng ban.
- Quản lý thông tin về căn hộ và cư dân.
- Tạo thống kê về doanh thu và hiệu suất hoạt động của công ty.

**Công nghệ:** C#, ADO.NET, SQL Server, Visual Studio, Window Form, EmguCV.
